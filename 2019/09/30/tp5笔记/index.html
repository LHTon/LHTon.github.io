php基本类型

四种标量类型：

  • boolean （布尔型）
  • integer （整型）
  • float （浮点型, 也称作 double)
  • string （字符串）

两种复合类型：

  • array （数组）
  • object （对象）

最后是两种特殊类型：

  • resource　（资源）
  • NULL　（NULL）

说一下资源类型：
资源 resource 是一种特殊变量，保存了到外部资源的一个引用。

资源类型变量保存有为打开文件、数据库连接、图形画布区域等的特殊句柄。

缓存Memcacheed

  • Memcacheed是高性能的分布式内存缓存服务器，通过缓存数据库查询结果，

    减少数据库的访问次数,提高Web应用的速度和扩展性
  • 为什么要使用Memcache
  1. 对数据库的高并发读写
  2. 对海量数据的处理
